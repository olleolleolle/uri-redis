URI-Redis v0.2

redis://host:port/dbindex

Usage

require 'uri/redis'

conf = URI.parse 'redis://localhost:8999/2'
conf.host                 # => localhost
conf.port                 # => 8999
conf.db                   # => 2

Installation

Get it in one of the following ways:

  • gem install uri-redis
  • git clone git://github.com/delano/uri-redis.git
